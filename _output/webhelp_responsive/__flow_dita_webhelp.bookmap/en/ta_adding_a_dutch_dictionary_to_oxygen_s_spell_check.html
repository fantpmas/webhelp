Adding a Dutch dictionary to Oxygen's Spell Check

If you use OxygenXML to write Dutch texts, you can add a Dutch dictionary to the spell checker.

Proceed as follows to install the Dutch dictionary in Oxygen:
  1. Locate the following file: S: \ temp \ hunspell_NL
  2. Copy both files to C: \ Program Files \ Oxygen XML Author 20 \ dicts \ spell.
  3. Restart OxygenXML.
  4. Make sure the language of your topics is set to NL, so for example: <task xml:lang="nl">.
  5. Do one of the following:
    If you prefer ...select ...
    Automatic spell check Options > Preferences > Spell Check > Automatic spell check
    Manual spell check Edit > Check Spelling or Edit > Check Spelling in Files
