DITA for kids (WEG?)

DITA for kids is a project teachers can make use of in class. The project itself is finished, but the code is open-source and can still be edited.

Teachers can download and edit Wikipedia articles with the open-source tool. All content in the articles (including images) can be reused and edited in the DITA-editor. The teacher can then group the new topics in themes and publish everything to a package. This package can then be shared with other PCs.

The full explanation can be found on http://dita.xml.org/node/2400.
